# `runbooks/` — Phase 13c

Phase 13a does not ship runbooks. Phase 13c (ch.13.12) adds:

- `runbook.md` — the page an on-call engineer opens at 3am. Ordered
  diagnostics. Dashboard links. Escalation steps.
- `on-call.md` — rotation policy, handoff, postmortem template.
- `dr-drill.md` — the scripted 30-minute regional-failover exercise.
- `chaos-gameday.md` — monthly Chaos Mesh fault scenarios run against the
  v2 platform.

Part 08 ch.02 (Backup & DR) + Part 11 ch.07 (Chaos engineering) are the
prerequisite reading.
